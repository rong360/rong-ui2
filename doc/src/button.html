# Button 按钮

# 基本用法

<Button type="warning" :radius="true" :fill="false" :btnStyle="btnStyle" @on-click="doClick">
		圆角 空心 示警 自定义宽度
</Button>

# Props

参数 说明 类型 可选值 默认值
type 按钮类型 string default warning disabled default
radius 按钮是否有圆角 boolean true false false
fill 背景是否填充 boolean true false true
btnStyle 自定义样式 object {}

# Events

事件名称 说明 回调参数
on-click 点击button回调函数 event

slot名称 说明

代码演示

<template>
  <div class="button-example">
    <Titlebar theme="b"
              @on-back="onBack">Button</Titlebar>

    <div class="title">default</div>
    <Button>提交</Button>

    <div class="title">空心默认圆角按钮（:fill="false" :radius="true"）</div>
    <Button :fill="false"
            :radius="true">提交</Button>

    <div class="title">示警按钮（type="warning"）</div>
    <Button type="warning">提交</Button>

    <div class="title">空心示警按钮(type="warning" :fill="false" :radius="true")</div>
    <Button type="warning"
            :fill="false"
            :radius="true">提交</Button>

    <div class="title">不可用按钮（type="disabled"）</div>
    <Button type="disabled">提交</Button>

    <div class="title">不可用按钮（type="disabled" :fill="false" :radius="true"）</div>
    <Button type="disabled"
            :fill="false"
            :radius="true">提交</Button>

    <div class="title">自定义（type="dark-yellow", 样式表里定义.r--button-dark-yellow相关样式 ）</div>
    <Button type="dark-yellow">提交</Button>
  </div>
</template>

<script>
export default {
  data () {
    return {}
  },
  methods: {
    onBack () {
      this.$router.push('/')
    }
  }
}
</script>
