# API

参数 说明 类型 可选值 默认值
title 标题文案 ReactNode
theme 标题栏背景主题 string a b c d e f a
showBackto 是否显示返回按钮 Boolean true
backArrowStyle 箭头自定义样式 CSSProperties
l 左侧插槽 ReactNode
r 右侧插槽 ReactNode
onBack 点击返回按钮回调 function

代码演示

const [age] = useState(18)

const title = useMemo(() => `标题${age}`, [age])
const r = useMemo(() => <strong>home</strong>, [])
const backArrowStyle = useMemo(() => ({ color: '#333' }), [])
const goHome = useCallback((e) => { history.push({ pathname: '/' }) }, [history])

return (
  <div>
    <Titlebar theme='b' onBack={goHome}>Titlebar</Titlebar>

    <Titlebar
      theme='a'
      title={title}
      r={r}
      showBackto={true}
      onBack={goHome}
      backArrowStyle={backArrowStyle}
    />

    <Titlebar theme='b' title='标题' />

    <Titlebar theme='c' title='标题' />

    <Titlebar theme='d' title='标题' />

    <Titlebar theme='e' title='标题' />

    <Titlebar theme='f' title='标题' />
  </div>
)
