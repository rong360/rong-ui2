# TextScroll

文字滚动效果

# 基本用法

# 组件引入

import { TextScroll } from 'rong-ui2';
1

# 调用

<TextScroll list={list}/>
1

# API

参数 说明 类型 可选值 默认值
list 文案列表 Array
prepend scrollbar前插槽 ReactNode
append scrollbar后插槽 ReactNode
